


# Automatic File Carving

> [!info] __File Carving__ 
>the practice of extracting files based on content, rather than on metadata
## Foremost

```
foremost -o (Path2Outdir) (path2image)
```

## Scalpel

__Config File__ `/etc/scalpel/scalpel.conf`

```
scalpel -o (path2Outdir) (path2image)
```

## Photorec

```
photorec \log \d (path2Outdir) (Path2image)
```
